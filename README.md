
# Release 1.0

```txt
Usage:
  ./image2ascii [OPTIONS]

Converts an image to ascii art.

Optional arguments:
  -h,--help             Show this help message and exit
  -v,--verbose          Increases verbosity
  -i,--input INPUT      Input image to convert (Currently only supporting PNG)
  -o,--output OUTPUT    Output file location (Defaults to output.txt)
  -d,--dimensions DIMENSIONS
                        (optional) Resize image for ascii output dimensions
                        [Example: -d 25x25]
```
